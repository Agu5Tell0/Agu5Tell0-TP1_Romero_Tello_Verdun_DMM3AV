Explicación

En este proyecto desarrollamos una landing page experimental pensada exclusivamente para escritorio, cuyo objetivo principal es permitir al usuario interactuar con dos imágenes superpuestas mediante el cursor. Utilizamos HTML, CSS y JavaScript con la biblioteca p5.js para lograr un efecto de “revelado” que deja ver la imagen inferior al pintar con el mouse. Además, incorporamos un <video> de fondo en bucle para aportar una atmósfera visual envolvente.

En el HTML estructuramos la página con etiquetas semánticas como <header>, <main> y <footer>, e incluimos un contenedor con el ID "canvas-container" donde se monta el canvas generado por p5.js. Vinculamos la hoja de estilos externa style.css, la fuente Inter desde Google Fonts, la librería p5.js desde CDN y nuestro script personalizado main.js. También colocamos un <video> con position: fixed y z-index: -1 para que funcione como fondo sin interferir en la interacción.

En el CSS nos aseguramos de que el video ocupe toda la pantalla con object-fit: cover y de que el <canvas> se muestre centrado y a un tamaño fijo (1300x731px). Usamos flexbox para organizar el contenido y dimos estilos generales como colores, márgenes y tipografía. También definimos la disposición de las imágenes de los integrantes en una grilla flexible.

El código JavaScript implementa toda la lógica del efecto visual. Primero, en preload(), cargamos dos imágenes: una blanca encima y otra negra debajo. En setup() creamos el canvas e inicializamos un buffer donde se almacena la imagen superior. En cada fotograma, draw() dibuja ambas capas, y si se activa un reinicio automático, genera una animación circular que vuelve a cubrir progresivamente la imagen inferior.

La interacción se maneja con funciones de mouse. Al arrastrar (mouseDragged()), se crea un degradado radial con destination-out, lo que borra parte del buffer y deja ver lo que hay debajo. Al soltar el mouse (mouseReleased()), se activa un temporizador que tras seis segundos reinicia la imagen. Esto se logra con la función resetCanvas() que lanza una animación suave de restauración.

Con este trabajo buscamos experimentar con el comportamiento del canvas en p5.js, explorando técnicas de enmascarado y animación que permitan una experiencia visual dinámica y creativa.
